1. Overview

ChiliopusNotify is the centralized notification service for the Chiliopus platform. It provides a unified API for sending and managing notifications across multiple communication channels.

The service supports:

Email notifications

SMS notifications

WhatsApp notifications

Push notifications

In-app notifications

The system uses asynchronous processing, queue-based delivery, and event-driven workflows.

2. Goals

The primary goals of ChiliopusNotify are:

Provide a unified notification REST API

Support multiple communication channels

Deliver notifications asynchronously

Provide template-based messaging

Support event-driven notifications

Provide tracking and monitoring

Integrate with Chiliopus applications

3. Scope

ChiliopusNotify will be used by:

ChiliopusCRM

ChiliopusLMS

ChiliopusPOS

Web applications

Mobile applications

It will function as the communication backbone of the Chiliopus ecosystem.
